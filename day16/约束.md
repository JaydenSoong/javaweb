# 约束
* 约束是添加到列上的，用来约束列的。
1. 主键约束（唯一标识）
    * 特点：非空、唯一、可以被引用
    * 当表的某一列被指定为主键时后，该列就不能为空，不能有重复的值出现。
    * 创建表时指定主键的方式有两种：
        ```
        // 指定 stu_id 是主键列
        CREATE TABLE stu(
	        stu_id numeric(4) PRIMARY KEY,
            stu_name VARCHAR(20)
        );
        // 指定主键列是 stu_id
        CREATE TABLE stu(
	        stu_id numeric(4),
            stu_name VARCHAR(20),
            PRIMARY KEY(stu_id)
        );
        ```
    * 在修改表时指定主键 ALTER TABLE stu ADD PRIMARY KEY(stu_id);
    * 删除主键：ALTER TABLE stu DROP PRIMARY KEY;
